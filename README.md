# python-sudoku
 
Program tworzy plansze do gry w Sudoku w trzech trybach trudności:
* łatwym
* normalnym
* trudnym

Każda plansza ma tylko jedno rozwiązanie.
GUI zostało wykonane z użyciem modułu tkinter.
